Install an app

  • Donwload the app folder (first download the zip; unzip it)
  • Move it to the "apps" folder (can be found where the index.html is)
  • Edit config.js (also in the same place as the index.html and apps folder) to: 
  • window.config = [
			"amgous"
			]
  • But replace YOUR APP HERE with the name of the folder you just downloaded/added to the apps folder
